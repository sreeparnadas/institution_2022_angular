FV (Rate, Nper, [Pmt], PV, [Type])

  • Nper = Number of periods
  • [Pmt] = Payment/period
  • PV = Present Value
  • [Type] = When the payment is made (if nothing is mentioned, it’s assumed that the payment has been made at the end of the period)/li>
